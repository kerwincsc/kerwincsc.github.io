变量篇
------------------------------------------------------------

- 变量自增的实现方法

  #. i=`expr $i + 1`;
  #. let i+=1;
  #. ((i++));
  #. i=$[$i+1];
  #. i=$(( $i + 1 ));


bg fg jobs
------------------------------------------------------------

#. :kbd:`Ctrl+z` 暂停命令到后台
#. `bg %num` 将指定序号的任务放在后台执行
#. `fg %num` 将指定序号的任务调到前台执行
#. `jobs` 列出当前任务控制表

如果你使用的是其它 shell, 比如 zsh, 有自带的 jobs 命令, 会使结果有所不同;

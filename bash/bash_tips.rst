BASH 小技巧
======================================================================

变量篇
------------------------------------------------------------

- 变量自增的实现方法

  #. i=`expr $i + 1`;
  #. let i+=1;
  #. ((i++));
  #. i=$[$i+1];
  #. i=$(( $i + 1 ));


sed
------------------------------------------------------------

- 追加到行

  - 追加...

    - 匹配行 ``sed -i '/pattern/a\append' filename``
    - 指定行 ``sed -i '2a\append' filename``

  - 插入

    - 匹配行 ``sed -i '/pattern/i\append' filename``
    - 指定行 ``sed -i '2i\append' filename``

  *-i* 写入到文件, 使之生效; *a* 追加到行; *\\* 如果有多个空格, 需要加;

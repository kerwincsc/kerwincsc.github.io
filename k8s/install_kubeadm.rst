安装 kubeadm
======================================================================

前期准备
------------------------------------------------------------

  - CentOS 7
  - 2 GB 及以上的内存
  - 2 核 CPU 及以上
  - 每个节点拥有独一无二的 主机名, [#MAC地址]_, 和 [#product_id]_
  - 确保网络的连通性
  - 某些端口需要被打开
  - 关闭 Swap;



.. rubric:: 脚注

.. [#MAC地址] ip link / ifconfig -a
.. [#product_id] cat /sys/class/dmi/id/product_uuid

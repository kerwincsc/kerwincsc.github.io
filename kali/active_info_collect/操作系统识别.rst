操作系统识别
======================================================================

- 操作系统识别技术

  - 种类繁多
  - 好产品采用多种技术组合

- TTL 起始值

  - Windows: 128 (65 --- 128)
  - Linux / Unix: 64 (1-64)
  - 某些 Unix: 255

  编程修改 TTL 值, 伪装操作系统, 误导扫描结果;

主动识别
------------------------------------------------------------

- python

  通过 scapy 对 Linux 和 Windows 机器发包, 并接收回包,
  判断回包的 TTL 值, 来大致判断系统类型;

- nmap

  - nmap -O 172.31.225.72  # 大写的字母 o, 不是数字 0;

- xprobe2

  - xprobe2 172.31.225.72  # 准头比较差


被动识别
------------------------------------------------------------

- IDS
- 抓包分析
- 被动扫描
- p0f

  - 结合 *ARP地址欺骗*, 识别全网 OS

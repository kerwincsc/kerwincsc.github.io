防火墙识别
======================================================================

- 通过检查回包, 可能识别端口是否经过防火墙过滤
- 设备多种多样, 结果存在一定误差

  +---+------+-------------------+-------------------+
  |   | Send | Response          | Type              |
  +---+------+-------------------+-------------------+
  | 1 | SYN  | No                | Filtered          |
  |   +------+-------------------+                   |
  |   | ACK  | RST               |                   |
  +---+------+-------------------+-------------------+
  | 2 | SYN  | SYN+ACK / SYN+RST | Filtered          |
  |   +------+-------------------+                   |
  |   | ACK  | No                |                   |
  +---+------+-------------------+-------------------+
  | 3 | SYN  | SYN+ACK / SYN+RST | Unfiltered        |
  |   +------+-------------------+ /                 |
  |   | ACK  | RST               | Open              |
  +---+------+-------------------+-------------------+
  | 4 | SYN  | No                | Closed            |
  |   +------+-------------------+                   |
  |   | ACK  | No                |                   |
  +---+------+-------------------+-------------------+


  根据此表, 使用 scapy 编写脚本测试防火墙是否开启;

.. _nslookup_basic:

nslookup
======================================================================

交互式执行
------------------------------------------------------------

- 直接输入 ``nslookup`` 进入, 输入域名得到最终的IP地址
- 交互式指令

  - 查询指定类型的记录

    - 只查 A 记录: ``set type=a`` 同 ``set q=a``
    - 只查 MX 记录: set type=mx
    - 查询所有记录: set type=any

      可能会出现一种名叫 spf 的记录, 此记录是文本记录, 用于反垃圾邮件;

  - 更改域名查询服务器

    - server dns-server-you-want

      如: ``server 8.8.8.8``

命令行执行
------------------------------------------------------------

::

   nslookup [要查询的记录类型] [域名] [DNS服务器地址]

- nslookup -q=any 163.com

=================
 CURL 的常见用法
=================

参数解析
========

- **-o , --output <file>**

  将输入写入文件, 不打印到标准输出;

  .. code-block:: shell

     curl -o aa www.baidu.com
     # 两个请求写入两个文件
     curl -o aa www.baidu.com -o bb www.163.com

- **-s, --silent**

  静音模式. 不显示进度条或者错误信息, 但仍在输出数据

- **-S, --show-error**

  一般与 **-s** 共同使用, 用于在静音模式中显示错误信息;

- **-f, --fail**
  
  如果服务器传输文件失败, 会返回一个 HTML 文件说明;
  不太明白, 暂时未遇到这种情况, 不过很多使用 bash
  脚本进行自动安装的软件会添加该选项, 建议加上;
